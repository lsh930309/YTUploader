# Binary Dependencies

이 디렉터리에는 배포 또는 로컬 실행에 필요한 Windows 바이너리를 둡니다.

- `losslesscut/` 디렉터리 또는 `LosslessCut.exe`를 포함한 런타임
- `mkvmerge.exe`
- `mpc-be64.exe`
- 또는 `mpc-be/` 디렉터리 전체
- 선택: `ffmpeg.exe`, `ffprobe.exe`

바이너리는 크기와 라이선스 이슈 때문에 저장소에 포함하지 않습니다. 각 도구의 공식 배포본을 내려받아 이 위치에 복사한 뒤 `build_script.py`를 실행하세요.
